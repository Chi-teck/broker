## Broker

A docker based template for web development.

### Prerequisites

1. Linux (AMD64)
2. Docker
3. [Task](https://taskfile.dev)

### Installation

1. Download the repository
2. Edit [Taskfile.yml](./Taskfile.yml)
3. Edit [Dockerfile](./image/Dockerfile)
4. Run `task rebuild`

### Usage

```
task start connect
```
